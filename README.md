# GameOfLife
### Controls:
| Key | Effect |
| ------ | ------ |
| spacebar | play/pause |
| c | clear board |
| g  |  show/hide grid |
| mouse buttons | toggle cells (can click + drag) |
| "[" and "]" | increment/decrement random % |
| r | randomise board |
| left/right arrows | step forward/back |
| up/down arrows | increment/decrement fps|
